# 📘 Assignment: Python Text Processing

## 🎯 Objective

Master Python string manipulation, file I/O operations, and text processing techniques. Build practical programs that read, process, and transform text data using Python's built-in string methods and file handling capabilities.

**Skills practiced:** String manipulation, file I/O, text processing, regex basics, data parsing

## 📝 Tasks

### 🛠️ String Manipulation and Analysis

#### Description
Write functions to analyze and manipulate text strings using Python's string methods. Create a program that performs various operations on user-provided text.

#### Requirements
Completed program should:

- Implement a function `count_words()` that counts words in a string
- Implement a function `reverse_string()` that returns a string in reverse order
- Implement a function `find_longest_word()` that identifies the longest word in a string
- Implement a function `convert_case()` that converts text to uppercase, lowercase, and title case
- Demonstrate each function with example text input

### 🛠️ File Reading and Writing

#### Description
Create a program that reads text from a file, processes the content, and writes results to a new output file.

#### Requirements
Completed program should:

- Read a text file line by line
- Count total lines, words, and characters in the file
- Write a summary report to an output file with these statistics
- Handle file not found errors gracefully with informative messages
- Strip leading/trailing whitespace from each line during processing

### 🛠️ Text Processing Application

#### Description
Build a practical text processing utility that combines string manipulation and file I/O to perform real-world text transformations.

#### Requirements
Completed program should:

- Read a text file containing a list of names or data
- Remove duplicates while preserving order
- Sort the entries
- Remove unnecessary spaces and special characters
- Write the cleaned data to an output file
- Display summary statistics (original count, final count, duplicates removed)
